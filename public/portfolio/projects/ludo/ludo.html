Crazy Ludo (飞行棋)

The Overview

Crazy Ludo is a modern, web-based implementation of the classic board game (known as 飞行棋 in China). Unlike traditional board games, this digital version focuses on a streamlined, offline-first experience with a custom-designed 52-grid track.

The project was built entirely from scratch using Vanilla JavaScript and the HTML5 Canvas API, demonstrating efficient rendering and game state management without the overhead of heavy game engines.

Key Features & Mechanics

Hybrid AI System

Supports mixed gameplay with both Human and AI players. The AI automatically handles dice rolling and move selection, allowing for single-player practice or filling empty slots.

Custom Track Logic

Features a 52-grid outer loop with corner bases. Implements classic "Jump" (same color +4) and "Eat" (capture opponent) mechanics for high-stakes interaction.

The game includes a 6-grid Victory Runway that requires precise dice rolls to finish, adding tension to the end-game phase. The starting mechanism aligns the launch pad with the track entrance, ensuring a smooth visual flow from base to board.

Technical Implementation

The core challenge was creating a responsive, procedurally drawn board that scales across devices.

  • Canvas Rendering: The board, including the complex "Hollow Plus" shape and individual grid cells, is drawn via JS functions like buildRing() and drawBoard(). This allows for infinite resolution scaling.
  • State Machine: A central S object manages the game state (Current Player, Dice Result, Tokens Position), enabling features like "Auto-skip" when no moves are available.
  • Dynamic UI: The "Roll Dice" button dynamically positions itself relative to the canvas to ensure accessibility on both mobile and desktop screens.
