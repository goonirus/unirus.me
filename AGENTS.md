# Codex Start Here

For this repository, read these before substantial work:

- `docs/codex-start-here.md`
- `docs/codex-collaboration.md`
- `docs/portfolio-workflow.md`
- `docs/project-state.md`

Key rules:

- Do not push remote unless the user explicitly asks in that moment.
- Use relative paths inside site files.
- Be careful with Chinese text and encoding.
- Ask before changing extra content, tone, project positioning, or structure that the user did not request.
- Directly fix obvious technical bugs such as broken links, wrong paths, mismatched cards, and image display issues.
- Do not add fake download or itch.io buttons. Links must point to real deployable targets.
