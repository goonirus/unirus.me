# 如何运行 home.html

有几种方式可以在本地运行这个网站：

## 方法 1: 使用启动脚本（推荐）

### Windows 批处理文件
双击 `start-server.bat` 文件，脚本会自动启动本地服务器。

### PowerShell 脚本
在 PowerShell 中运行：
```powershell
.\start-server.ps1
```

服务器启动后，在浏览器中访问：**http://localhost:8000/home.html**

## 方法 2: 手动使用 Python

如果你已经安装了 Python，在项目目录下运行：

**Python 3:**
```bash
python3 -m http.server 8000
```

**Python 2:**
```bash
python -m http.server 8000
```

然后在浏览器中访问：**http://localhost:8000/home.html**

## 方法 3: 使用 Node.js

如果你安装了 Node.js，可以运行：

```bash
npx http-server -p 8000
```

然后在浏览器中访问：**http://localhost:8000/home.html**

## 方法 4: 使用 VS Code Live Server

1. 在 VS Code 中安装 "Live Server" 扩展
2. 右键点击 `home.html` 文件
3. 选择 "Open with Live Server"

## 方法 5: 直接打开（不推荐）

虽然可以直接双击 `home.html` 在浏览器中打开，但由于使用了 CDN 资源，建议使用 HTTP 服务器运行以确保所有功能正常工作。

---

**提示**: 服务器启动后，按 `Ctrl+C` 可以停止服务器。
